# Qt로 과목별 필기장 만들기

Qt에서 배운 `QMainWindow`, `QWidget`, `QComboBox`, `QTextEdit` 등을 활용해  
과목별로 내용을 저장하고 다시 수정할 수 있는 간단한 필기장 프로그램을 만들어보았다.

이 프로그램은 사용자가 과목을 선택한 뒤, 각 과목에 맞는 메모를 작성하고 다시 불러올 수 있도록 구성했다.  
단순히 메모를 입력하는 기능만 만드는 것이 아니라, 과목별로 내용을 나누어 관리할 수 있도록 구현하는 데 초점을 두었다.

## 이 프로그램은 다음과 같은 흐름으로 동작한다

- 메인 창에서 과목 선택
- `Open` 버튼으로 필기장 창 열기
- 과목별 메모 작성
- 창을 닫고 다시 열어도 과목별 저장된 내용 유지
- `수정` 버튼으로 다시 편집 가능
- 수정 후 저장하지 않고 나가면 이전 데이터 유지

## 프로그램 구성

이번 프로그램은 `MainWindow`와 `widget` 클래스로 나누어 구성했다.  
`MainWindow`에서는 과목 선택과 필기장 창 열기를 담당하고,  
`widget` 클래스에서는 실제 메모 입력과 저장, 수정, 닫기 기능을 담당하도록 만들었다.

메인 화면에서는 `QComboBox`를 사용해 과목을 선택하고,  
`QLineEdit`를 통해 현재 선택된 과목을 확인할 수 있도록 했다.  
이후 `Open` 버튼을 누르면 필기장 창이 열리고, `Close` 버튼을 누르면 프로그램이 종료된다.

필기장 화면에서는 `QTextEdit`를 사용해 메모 내용을 입력하고,  
저장, 수정, 닫기 버튼으로 메모 상태를 제어할 수 있도록 구현했다.

## 구현 방식

과목 선택은 메인 창에서 이루어지며, 사용자가 선택한 과목에 따라  
필기장 창에서 보여줄 메모 내용도 함께 바뀌도록 만들었다.

각 과목의 메모는 문자열 배열을 활용해 구분해서 저장했고,  
현재 선택된 과목의 인덱스를 기준으로 해당 위치의 메모를 불러오거나 저장하는 방식으로 처리했다.

필기장 창을 열면 현재 선택된 과목에 해당하는 메모 내용을 다시 불러와  
이전에 작성한 내용을 이어서 확인하거나 수정할 수 있도록 했다.

또한 저장 버튼을 누르면 현재 작성한 내용을 해당 과목 메모에 반영하고,  
수정 버튼을 누르면 다시 편집할 수 있도록 구성했다.


## 문제 해결

구현하면서 확인한 문제 중 하나는  
한글 입력 중 저장 버튼을 바로 누를 경우 조합 중이던 마지막 글자가 반영되지 않는 현상이었다.

이 문제를 해결하기 위해 저장 전에 포커스를 다른 위젯으로 넘겨  
입력 상태를 먼저 확정시키는 방식을 사용했다.  
이렇게 하면 조합 중인 글자도 정상적으로 입력창에 반영된 뒤 저장할 수 있었다.

## 느낀점

간단한 실습이었지만 Qt의 기본 구조와 사용 방식에 익숙해지는 데 도움이 되었던 시간이었다.

직접 위젯을 배치하고 기능을 연결해보면서  
`signal-slot` 구조와 UI 구성 방식에 대해 조금 더 익숙해질 수 있었다.

또한 단순한 메모 기능이라도  
과목별로 데이터를 어떻게 나누어 저장하고 다시 불러올지 고민해보면서  
상태를 관리하는 방식의 중요성도 함께 느낄 수 있었다.
