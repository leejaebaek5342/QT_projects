# QT

이 저장소는 Qt를 활용해 진행한 프로젝트들을 정리하고, 개발 과정과 결과물을 기록하기 위한 공간입니다.

## 목적

- Qt를 활용한 프로젝트 결과물 관리
- 프로젝트별 개발 기록 정리
- 구현 과정과 학습 내용 보관

## 구성

현재 저장소에는 다음과 같은 Qt 기반 프로젝트들이 포함되어 있습니다.

- `QT_MEMO` (간단한 필기장)
- `QT_Attendance_Manager` (출석관리 프로그램)
- `ERP` (재고관리 프로그램)

앞으로 새로운 Qt 프로젝트를 계속 추가하며, 각 프로젝트의 개발 기록도 함께 정리해 나갈 예정입니다.

## 기술 스택
- Language: C++
- Framework: Qt
- Build: CMake
